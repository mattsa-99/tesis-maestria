# 1. Introducción
El fraude financiero ha evolucionado hacia estructuras de red complejas. Esta tesis propone ir más allá del análisis transaccional punto a punto, utilizando la "forma" de los datos (topología) para detectar comportamientos ilícitos.

## 1.1 Problema
Los sistemas actuales fallan ante ataques coordinados de múltiples cuentas que imitan comportamiento humano pero mantienen una estructura de grafo anómala.

## 1.2 Objetivos
- Implementar complejos de Vietoris-Rips para extraer rasgos topológicos.
- Clasificar transacciones mediante grafos dirigidos.
